了解复盘镜

复盘镜（WorkMirror）是一个 Windows 托盘常驻的本地回顾工具：自动收集你电脑上的工作内容（应用使用、代码修改、网页浏览），整理成可追溯的工作记录，并生成日报/周报与技能变化。

你能得到什么

  • 每天/每周的回顾：每条结论都能点开看到数据来源与明细。
  • 技能树与能力变化：增长来源可追溯到贡献的工作记录。
  • 离线也能用：关闭 AI 时可完全离线，规则版摘要与回顾不依赖外部 AI。
  • 遇到"空白页/没数据"也不用困惑：状态页会告诉你问题在哪里，并给出下一步怎么修。

隐私与本地优先

  • 数据默认写入本地 SQLite（默认 ./data/workmirror.db），不做云同步/账号体系。
  • 本地 HTTP 仅监听 127.0.0.1，端口随机分配。
  • 启用 AI 时仅发送脱敏后的最小必要文本，用于生成摘要与建议。

它以什么形式运行（v0.2.0-alpha.1）

  • workmirror.exe（Windows 10/11 托盘常驻）
  • 内置本地 HTTP 服务（随机端口，仅监听 127.0.0.1）
  • 内置 Web 界面（React/Vite SPA，优先加载同目录 frontend/dist/，否则回退内嵌资源）
  • 端口发现：启动后写入 ./data/http_base_url.txt

它怎么工作（简述）

采集 → 生成工作记录（切分/聚合/摘要）→ 汇总到技能与回顾 → 每条结论可点开看到数据来源。

目前不包含什么

  • 云同步/账号体系/团队协作
  • macOS/Linux（当前阶段）
  • 复杂目标管理（TODO/OKR/习惯）
