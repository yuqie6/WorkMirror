常见问题

需要联网吗？

不强制。关闭 AI 时不需要联网；启用 AI（包括内置免费模型或自配供应商）后，仅在需要生成摘要/建议时调用外部服务。

为什么页面为空/没有数据？

  • 先打开「运行状态」页查看各项服务是否正常运行，以及最近是否有错误。
  • 代码监控需要先配置监控目录（在设置页添加你的项目文件夹路径）。
  • 浏览器监控依赖本地浏览历史文件；可在设置中查看自动探测结果并手动指定。

支持 macOS/Linux/手机吗？

当前阶段只支持 Windows 10/11，且不提供云同步/账号体系。

如何导出诊断信息？

在「运行状态」页点击"导出诊断包"按钮，会下载一个脱敏后的 zip 文件，可用于排查问题或联系支持。
