FAQ

Do I need internet?

Not required. Without AI configured, WorkMirror works completely offline using rule-based summaries. With AI enabled (built-in free service or your own API key), external services are only called when generating summaries and suggestions.

Why is the page empty/no data?

  • First, open the "Status" page to check if services are running normally and if there are any recent errors.
  • Code monitoring requires configuring monitored folders (add your project folder paths in Settings).
  • Browser monitoring depends on local browser history files; check auto-detection results in Settings and specify manually if needed.

Does it support macOS/Linux/mobile?

Currently only Windows 10/11 is supported, with no cloud sync or account system.

How to export diagnostic info?

On the "Status" page, click "Export Diagnostics" button to download a sanitized zip file for troubleshooting or contacting support.
